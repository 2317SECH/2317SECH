## 🏗️ Lo que estoy construyendo

**BIMO** es mi ecosistema propio de productos tecnológicos. La idea central: combinar **inteligencia artificial, automatización y desarrollo de software** para crear soluciones escalables, no proyectos aislados — cada pieza (trading, e-commerce, agentes de IA) comparte la misma disciplina de ingeniería y crece bajo una sola raíz.

<br/>

## 🤖 Proyectos

### 🧠 BIMO
Ecosistema principal — la base técnica que sostiene todo lo demás.

- Agentes de IA autónomos
- Automatización de procesos de negocio
- Diseño y construcción de APIs
- Dashboards internos de operación
- Backend y arquitectura de software
- Herramientas inteligentes a medida

<br/>

### 📈 BIMO Copy
Sistema de **trading algorítmico** construido de cero.

- Bots de ejecución automática
- Gestión de riesgo programática
- Backtesting de estrategias
- Análisis estadístico de mercado
- Integración con brokers vía API
- Automatización de estrategias de trading

<br/>

### 🛍️ BIMO Shop
Tienda construida sobre **Shopify**, con foco en producto y marca, no solo en catálogo.

- Shopify + Storefront API
- GraphQL para consumo de datos en tiempo real
- Automatización de catálogo y curaduría
- Integraciones con proveedores y logística
- Diseño de experiencia de usuario (UX)
- Construcción de marca e identidad
